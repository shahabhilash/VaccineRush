# VaccineRush

A simple endless runner game built using Unity where the player controls an injection and must move up and down to avoid incoming virus obstacles.

---

## 🎮 Gameplay

- Control the injection movement (up & down)
- Avoid virus obstacles
- Survive as long as possible
- Game speed may increase over time

---

## 🛠️ Tech Stack

- Unity (Game Engine)
- C#
- Basic Physics & Collision System

---

## 🚀 Features

- Simple and responsive controls
- Endless gameplay loop
- Obstacle spawning system
- Lightweight and beginner-friendly project
